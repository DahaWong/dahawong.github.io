成为「部落客」

这周花了些时间建立了我的个人博客。如你所见，基本框架已经成型了。为什么要建博客？

拒绝互联网壁垒

Blog，也就是 Web Log。台湾称其为「部落格」，大陆译作「博客」。一年前2017我开了一个微信公众号，打算在上面写点文字。后来我放弃了更新，原因有：

  1. 编辑不方便，不支持 markdown 语法，要使用第三方的软件进一步排版。
  2. 不允许插入公众号以外的链接。
  3. 评论区不能即时呈现。
  4. 必须手机与电脑结合操作。
  5. 无法用任意的搜索引擎搜索到公众号内容。
  6. 不能提供除微信外的其他订阅方式。

除了第一点和第四点之外，其他都是 「在墙内再添一道墙」的行为。不只是微信，各大网站不约而同地发出这样的信息：来我们这里，别去他处了。然而信息的价值依靠传播实现，任何建造壁垒的行为都是违背互联网精神的。如李如一先生在《告別微信》中说的：

我反对把自己文化生活的主导权让渡给任何一个品牌的生活方式。

我拒绝一个不允许链接到自有生态圈以外资源的媒体基础设施。

走出舒适区

我认为走出舒适区有两种实现方式：建立新习惯与抛弃旧习惯。正好不久前看到了 Joway 的博客，就在想自己能不能也搭建一个。于是我把建博客作为本周的目标 ，尝试建立新的习惯。

那么，如何食用？

阅读

  • 应用了响应式设计，所以用什么设备阅读都可以。

  • 横屏终端阅读体验更佳。

  • 出于安静阅读的初衷，滚动过程中页面字体颜色会减淡一些。

评论

  • 勾选「匿名评论」Comment·Anonymously即可获得隐身风衣。
  • 如果想要实名ID聊天，先在 Commento 注册一下，这样能在邮箱里接收新回复。

订阅

  • RSS 订阅，下拉滚动本页面唤出导航栏，点击右侧的即可复制地址，粘贴到 RSS 阅读器订阅即可。
  • 邮箱订阅，感谢海德薇平台提供的服务。
  • 也可以通过电报频道接收电波。里面包含文章更新，附带一些生活和设计的遐思。（我做了一个本站的随读Instant·View模板，以便在电报里瞬间打开。）

怎么建个人博客

你可以使用现成的静态网页生成引擎 Hexo。在它的用户社区 ，有各式各样的主题和插件供你使用。如果了解一些网页编程的知识，自己开发也很方便。
