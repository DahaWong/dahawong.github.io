任务自动化最迷人的一点在于，当一切配置安顿好之后，只需要少许触发条件，「机器人」就悄悄地帮你完成了一切。你可能甚至意识不到它们的发生，自然而美好。

Tasker 是 Android 系统最优秀的任务自动化软件。它充分地利用手机的潜能（比如各类传感器）检测你当前所处的状态并判断你的意图，从而完成任务。而作为使用者，我们可以不需要编写程序，在交互界面做好配置即可做到。下面罗列一些常用的应用场景：

播放音乐

非常简单也令人愉悦的配置：「当连接上指定的蓝牙设备时，开始播放音乐」。我主要使用的音乐播放设备是 Airpods 和蓝牙小音箱。戴上 AirPods、AirBattery 检测到后帮它连上蓝牙、音乐自然地流淌出来，音量正合适。

我喜欢睡前戴耳机聆听着音乐入睡。因为夜深人静，这个时候只需要很低的音量。所以便引入了这样的配置：「到了睡觉时间，调低媒体音量，30分钟后静音、终止音乐软件播放」。

睡眠记录

说到睡眠，睡眠时间记录也是常用的场景。这里配合了 NFC 标签的使用，我把 NFC 标签贴在床头，每当到了睡觉时间 Tasker 唤起 NFC ，将手机靠近标签，这时 Tasker 里提前定义好的「入睡时间」变量便会记录下此刻的时间；早上清醒之后，再打一次卡，记录「起床时间」变量，后台计算出睡眠时间后，告诉我晚上睡了多久，是否满 8 个小时。同时添加一条记录到「睡眠记录」文件，每个周五的时候提醒我查看这周的睡眠状况。仅仅使用了一些变量，就可以轻松记录睡眠。更重要的是用了 NFC 标签这一物理打卡器，一般不会忘记。

签到打卡

这个功能需要配合 Nova Launcher 启动器的 Swipe Up 和 AutoInput 软件。Nova Launcher 的 Swipe Up 支持软件图标上滑触发 Tasker 任务。所以我配置了上滑图标自动签到相关软件，Tasker 负责打开软件，AutoInput 负责自动完成点击任务。

尾声

使用 Tasker 还可以完成其他一些小任务，比如：步数达到一个阈值时，晚些时候提醒拉伸放松；到家离家时问候以及提醒待办事项；电源不足提醒充电，充满电时提醒完成；打开 Instapaper 时启动横屏支持（我习惯横屏阅读）；话费不足时打开充值界面 …… 发挥想象力，一切皆有可能。
